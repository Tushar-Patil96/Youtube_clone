# Youtube_clone
This YouTube Clone is a web-based application designed using HTML, CSS, and JavaScript that replicates the basic layout and functionality of YouTube. It features a responsive user interface with a video grid, navigation bar, and video preview sections. The project focuses on front-end design and UI/UX practice, making it suitable for learning modern web development concepts.

Technologies Used:
#HTML
#CSS
#JavaScript

Features:
1. YouTube-like homepage layout
2. Responsive design
3. Video thumbnails and sidebar navigation
4. Clean and modern UI

How to Run the Project:
1. Download or clone the repository
2. Open the project folder
3. Open index.html in any web browser
